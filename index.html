Termino.js - The Ultimate Command-Line App Builder for Browser and Process-Based Environments

Create terminals in the browser with MarketingPipeline's Termino.js library - great for making terminal animations, terminal like apps & terminal based games.

  • Super easy to use
  • Supports both browser and process-based environments
  • Highly customizable to your own liking / needs (bring your own CSS / HTML - edit your terminal to your style)!
  • Await / promise-based inputs & more
  • Add your own features / functions (including functions on keys or click events!)
  • Lightweight - no dependencies
  • Create as many terminals you want!
  • and LOTS more!

A JavaScript library made by @MarketingPipeline. Source code is available on GitHub. License can be found here.

Termino.js is a powerful JavaScript library that allows you to build command-line apps for both the browser and process-based environments. With Termino.js, you can easily create custom interfaces and workflows that are familiar to command-line users, making it an ideal solution for a wide range of projects.
